A Little About Me...


My name is Justin Schaaf, and I've spent the last 4 years of my life in front of a computer. I started programming when I was 11 and my elementary school offered a code.org programming club, in which I learned basic block programming. Ever since, I've loved to code.


I taught myself Java when I was in 6th grade, as I made Minecraft mods in my free time. In 7th grade, I took a Computer Explorations class, in which I learned how to create projects in Scratch. I also began attending Code Central, where I properly learned JavaScript and HTML. Also at Code Central, I experimented in GameMaker Studio, Angular (which this website is built in), and Ionic, the latter two I have grown to love.


Now, I'm developing crazier and crazier projects. I've made a program that locally runs a chat bot based on a config file, a website exclusively filled with colors, and more insane projects to come.
